# Project rules — sentinel-lua

- Never add `Co-Authored-By` trailers or any self-attribution to commits.
- `reference/` is git-ignored scratch space for reference material; never commit its contents.
- Comments stay minimal. Write none when the code is self-evident. When the intent isn't obvious, a single brief line explaining WHY — never restate what the code does, never block comments, no yandere-dev narration.
